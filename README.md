# AI Search Algorithms

Implementation of uninformed search algorithms applied to maze solving 
and adversarial game playing.

## Projects

### 1. Maze Solver — BFS, DFS, and Iterative Deepening Search
Three agents solve the same maze using different search strategies.

**Algorithms implemented:**
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Iterative Deepening Search (IDS)

**Output per agent:**
- Optimal solution path (exact steps)
- Number of nodes visited during search

### 2. 3D Connect 4 AI Agent
An agent that plays 3D Connect 4 using adversarial search.
